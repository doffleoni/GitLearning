
git diff показывает разницу между текущим неодслеживаемым состоянием и последним снимком репозитория.
git diff --staged показывает разницу между текущим отслеживаемым состоянием и последним снимком репозитория.
git diff COMMIT_ID показывает разницу между текущим состоянием репозитория и указанным снимком репозитория.
